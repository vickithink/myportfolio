

# Vicki — Freelance Portfolio Website

A modern, minimal, conversion-focused portfolio for a Full Stack Web Developer, built with React + Vite + Tailwind CSS.

---

## 1. Navbar
- Sticky top navigation with "Vicki" logo text
- Links: Home, Services, Portfolio, About, Contact (smooth scroll)
- Dark/Light mode toggle button
- Highlighted "Hire Me" CTA button
- Social icons (GitHub, LinkedIn, Instagram, X) linking to @vickithink

## 2. Hero Section
- Bold headline: *"I Build Modern, Fast & Scalable Websites for Your Business"*
- Subheadline about helping startups, schools, and small businesses
- Credibility line: Full Stack Developer | React & Node.js Specialist
- Three CTA buttons: Get a Free Consultation, View My Work, WhatsApp Me
- WhatsApp displayed prominently

## 3. Services Section
- Four clean cards: Business Website Development, Full Stack Web Apps, Frontend & UI Development, Authentication & Backend Setup
- Each card with icon, title, and bullet points of offerings

## 4. Tech Stack Section
- Minimal grid/icon display of technologies: React, TypeScript, Node.js, Express, MongoDB, Firebase, Git/GitHub

## 5. Portfolio / Case Studies
- Project cards with: name, problem, solution, result/impact, tech used, Live Demo & GitHub buttons
- Placeholder projects with realistic business-focused descriptions ready for Vicki to customize

## 6. Why Choose Me Section
- Six benefit cards: Clean design, Fast delivery, Clear communication, Secure solutions, Affordable pricing, Long-term support

## 7. Work Process Section
- Five-step visual timeline: Requirement Discussion → Planning & Design → Development → Testing → Delivery & Support

## 8. About Section
- Professional intro paragraph focused on helping businesses
- Highlights: Full Stack Developer, React & Node.js experience, clean/scalable apps, client-focused approach

## 9. Contact Section
- Headline: *"Let's Build Something Powerful Together"*
- Contact details: email, phone, WhatsApp, website, location
- Contact form with Name, Email, Project Details, and submit with success toast
- Floating WhatsApp button (visible site-wide)

## 10. Footer
- Social links (@vickithink), domain iamvicki.in, copyright text, theme-aware styling

---

## Design & Technical Details
- **Default dark theme** with purple/blue accent color palette
- Dark/light mode toggle using class-based switching
- Mobile-first responsive design throughout
- Smooth scroll navigation between sections
- Subtle entrance animations on scroll
- SEO meta tags in index.html
- Clean component architecture (one component per section)
- No backend needed — purely frontend, static portfolio
